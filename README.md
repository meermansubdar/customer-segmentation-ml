# Customer Segmentation Using Clustering

## 📌 Project Overview
This project performs customer segmentation using machine learning clustering techniques. It groups customers based on their behavior and spending patterns.

## 🚀 Features
- K-Means Clustering
- Elbow Method for optimal clusters
- PCA Visualization
- RFM Analysis (Industry-level)
- Silhouette Score evaluation
- Interactive Streamlit Dashboard
- CSV Download functionality

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Streamlit

## 📊 Datasets
- Mall Customer Dataset
- Online Retail Dataset

## ▶️ How to Run
1. Install dependencies
2. Run:
   streamlit run app.py
3. Upload dataset
4. View segmentation results

## 📈 Output
- Customer segments
- Visual graphs
- Downloadable CSV files

## 🎯 Conclusion
This project demonstrates how machine learning can be used to analyze customer behavior and improve business decisions.
